# TaskFlow – Project Management Tool

TaskFlow is a web-based project management application built using HTML, CSS, JavaScript, Node.js, and Express.

## Features
- Add tasks
- Set priority
- Mark tasks completed
- Delete tasks
- Dashboard statistics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express

## Run

npm install
node server.js

Open:
http://localhost:3000


# E-Commerce Website

A responsive and user-friendly E-Commerce Website developed to provide a seamless online shopping experience. Users can browse products, view details, add items to cart, and manage purchases through an intuitive interface.

## Features

- Home Page
- Product Listing
- Product Details
- Add to Cart
- Remove from Cart
- Responsive Design
- Modern UI
- Checkout Interface

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Storage:
- JSON / Local Storage

## Project Structure

ecommerce-website/

├── public  
│ ├── index.html  
│ ├── products.html  
│ ├── cart.html  
│ ├── style.css  
│ └── script.js  

├── server.js  
├── package.json  
├── README.md  

## Installation

Clone repository:

git clone <repository-url>

Move into folder:

cd ecommerce-website

Install dependencies:

npm install

Run project:

node server.js

Open browser:

http://localhost:3000
